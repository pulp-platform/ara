operand_queues_stage — Instantiate the in-lane operand queues

Author: Matheus Cavalcante License: Solderpad Hardware License v0.51 Description: This module contains the operand queues used in Ara’s vector lanes. It manages operand buffering and dispatching to the appropriate functional units (VFUs) such as ALU, MFPU, STU, SLDU, and Mask Unit. Each operand queue is a parameterized instance of operand_queue.
